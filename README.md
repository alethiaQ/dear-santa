## Installation and Usage

First, clone down this repo- frontend and API included within. Then run the following in your terminal: 
    
    $ cd dear-santa-api
    $ bundle install
    $ rails s -p 3001
    $ cd .. 
    $ cd frontend 
    $ npm install 
    $ npm start
